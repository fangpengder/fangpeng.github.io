Mac制作MacOS-U盘启动

发表于 2018-06-15 | 分类于 Mac OS

app源制作U盘启动方式

  1. App Store下载Mac OS
  2. 格式化U盘为Mac OS X扩展 (日志式)格式,GUID分区图,名称”Sierra”

dmg源制作U盘启动方式

  1. 格式化U盘为Mac OS X扩展 (日志式)格式,GUID分区图,名称”Sierra”
  2. 双击dmg，会磁盘工具里显示相应磁盘镜像
  3. 选分好区的U盘,点击恢复,恢复来源选择刚挂载的镜像
